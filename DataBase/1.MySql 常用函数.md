### 数学函数

| 函数          | 功能                        |
| ------------- | --------------------------- |
| ABS(x)        | 返回 x 的绝对值             |
| CEIL(x)       | 返回大于 x 的最小整数值     |
| FLOOR(x)      | 返回小于 x 的最大整数值     |
| MOD(x,y)      | 返回 x/y 的模               |
| RAND()        | 返回 0～1 内的随机值        |
| ROUND(x,y)    | x 四舍五入，保留 y 位小数点 |
| TRUNCATE(x,y) | x 截断位 y 位小数           |
